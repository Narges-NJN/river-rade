#### river-rade
Move your plane with arrow keys or A-D keys


##### To install Ncurses library(on debian based):
```shell
sudo apt-get install libncurses5-dev libncursesw5-dev
```
##### To compile and run:

```shell
gcc river_rade.c -lncurses -o river_rade.out
```

```shell
./river_rade.out
```
